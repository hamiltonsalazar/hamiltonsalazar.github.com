DESARROLLO WEB

¿Qué es un servidor Web?

Es una máquina de alto procesamiento (genralmente) que funciona como unidad central dentor de un sistema cliente/servidor. Este equipo contiene todos los archivos y servicios disponibles para los clientes.

¿Qué es un cliente Web

  • Es el equipo de computo que consume los servicios de manera remota al servidor o quién visualiza las páginas.
  • Todo aquel que se conecta al servidor.

¿Qué es hosting?

Es quien nos provee o alquila el sistema para almacenamiento de información, espacio para imágenes, vídeos y demás. Especialente requerimos de un hosting cuando construimos páginas que podrían ser muy visitadas y con un alto contenido.

¿Qué es un dominio?

Es el nombre único que se le asigna a cada página de Internet. Normalmente se usa para ser un direccionamiento más entendible y fácil de recordar para el ser humano en comparación con la dirección IP única que identifica a esta misma.

¿Qué es HTML?

Es el lenguaje de marcado o lenguaje de etiquetas el cual es usado para crear páginas web. Las siglas en inlgés significan: "HyperText Markup Language"

¿Qué es CSS?

Es el lenguaje de diseño gráfico o de estilo para aplicar a las páginas web, el cuál se aplica a los archivos en lenguaje de marcado (HTML).

¿Qué es Javascript?

Es el lenguaje de programación interpretado, el cual se encarga de toda la parte funcional en una página web.
